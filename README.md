## 📌 Problem Statement

Women in urban India face elevated safety risks, particularly
at night and in low-surveillance areas. Emergency response
systems are fragmented, slow, and not connected to community
organizations like NGOs.

---

## 💡 Solution

RakshaDeep is a real-time AI safety system that allows women
to report distress situations instantly, notifies nearby NGOs
through a live admin dashboard, and visualizes alert hotspots
on an interactive map — all with full privacy protection.

---

## ✨ Key Features

- 🚨 **Real-time alert monitoring** — instant distress reporting
- 🗺️ **Map-based NGO visualization** — shows nearest help centers
- 🔐 **Anonymous code-word alerts** — privacy-first panic trigger
- 📊 **Live admin dashboard** — status tracking and case management
- 🔒 **Firebase integration** — low-latency real-time database sync

---

## 🏗️ Architecture
User App (Streamlit)
↓
Firebase Realtime Database
↓
Python ML Backend
↓
Alert Dispatcher → NGO Network + Emergency Contacts
